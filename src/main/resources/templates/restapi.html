General

Reseptipankki is a database which contains recipes and categories for the recipes. There is a One-to-Many link between category and recipe. Recipe belongs to one category. Recipe contains the following attributes:

  • recipeId (long)
  • header (String)
  • comment (String)
  • rating (float)

Category contains the following attributes:

  • categoryId (long)
  • categoryName (String)
